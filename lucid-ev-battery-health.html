Lucid Battery Health

Powered by python-lucidmotors API

Last analyzed: Unofficial · Air & Gravity

📡 How to Pull Your Data (3 Steps)

1

Install python-lucidmotors

Requires Python 3.9+. Run this command in your terminal:

pip install lucidmotors

Don't have Python? Download it free at python.org

2

Run the fetch script

Save this as fetch_lucid.py and run it with: python fetch_lucid.py

import asyncio, json, getpass
from lucidmotors import LucidAPI

username = input("Lucid account email: ")
password = getpass.getpass("Password: ")

async def main():
    async with LucidAPI() as lucid:
        await lucid.login(username, password)
        v = lucid.vehicles[0]
        data = {
            "vin": v.config.vin,
            "model": v.config.model.name,
            "variant": v.config.variant.name,
            "battery_state": {
                "remaining_range": v.state.battery.remaining_range,
                "charge_percent": v.state.battery.charge_percent,
                "kwhr": v.state.battery.kwhr,
                "capacity_kwhr": v.state.battery.capacity_kwhr,
                "battery_health_level": v.state.battery.battery_health_level,
                "max_cell_temp": v.state.battery.max_cell_temp,
                "min_cell_temp": v.state.battery.min_cell_temp,
                "unavailable_range": v.state.battery.unavailable_range,
                "battery_pack_type": v.state.battery.battery_pack_type.name,
                "battery_cell_type": v.state.battery.battery_cell_type.name,
            },
            "charging_state": {
                "charge_rate_kwh_precise": v.state.charging.charge_rate_kwh_precise,
                "charge_limit_percent": v.state.charging.charge_limit_percent,
            },
            "chassis_state": {
                "odometer_km": v.state.chassis.odometer_km,
            }
        }
        print(json.dumps(data, indent=2))

asyncio.run(main())

Your credentials are never sent to this app — the script runs entirely on your machine.
